# 📊 DIP Lab — Image Enhancement System

An interactive **Digital Image Processing (DIP) web application** that demonstrates fundamental image enhancement techniques with real-time visualization and histogram analysis.

---

## 🚀 Features

✨ **Core Image Processing Operations**
- Grayscale Conversion (Luminance-based)
- Salt & Pepper Noise Injection
- Median Filtering (5×5 kernel)
- Histogram Equalization

📊 **Advanced Visualization**
- Input vs Output image comparison
- Real-time histogram plotting (R, G, B, Luminance)
- Overlay histogram comparison
- Histogram difference visualization

📈 **Statistical Analysis**
- Mean (Brightness)
- Standard Deviation (Contrast)
- Entropy (Information content)
- Skewness (Histogram distribution)
- Peak intensity analysis

🧪 **Pipeline Tracking**
- Step-by-step processing stages:
  - Load → Gray → Noise → Denoise → Enhance

---

## 🖼️ Demo Workflow

1. Upload an image (JPEG / PNG / BMP)
2. Apply processing steps:
   - Convert to grayscale
   - Add noise
   - Remove noise (median filter)
   - Enhance contrast (histogram equalization)
3. Analyze results using:
   - Histograms
   - Metrics table
   - Visual comparison

## 📊 Image Processing Concepts Used

### 1. Grayscale Conversion
Y = 0.299R + 0.587G + 0.114B


### 2. Salt & Pepper Noise
Randomly sets pixels to black or white.

### 3. Median Filtering
Replaces each pixel with the median value of its neighborhood (5×5 kernel).

### 4. Histogram Equalization
Enhances contrast using cumulative distribution function (CDF).
